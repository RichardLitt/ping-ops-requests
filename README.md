# ping-ops-requests

> Ping ops-requests

## Install

```
npm i -g ping-ops-requests
```

## Usage

```
ping-ops-requests --test -h hashhh -d domain.test
# For reals
ping-ops-requests -h Qm9280779829793794792 -d orbit.chat
```

You'll need a `NODE_GITHUB_ISSUE_BOT` token from GitHub to create issues.
